# Configs

My personal dotfiles for vim, tmux, qtile wm, and a few other programs. While I use the default setup for most applications, I have made a few customizations here to suit my workflow across different platforms. Feel free to use any of the configurations here, but keep in mind that you might need to adjust them to fit your specific needs and preferences.  

Since this is a personal repository, documentation is minimal (and will likely stay that way), but I have included a few comments here and there to explain the purpose of different sections and settings.
